# 网页介绍

[静态网页展示](https://atwelve12.github.io/Yorushika.github.io/)

这是有关一个乐队"ヨルシカ"（Yorushika）专辑和歌曲的网页.

通过接入网易云音乐内嵌播放器实现选择并播放音乐.

## 问题:

音乐版权问题：使有些音乐无法嵌入播放器
音乐加载问题：打开网页时尽量不使用vpn`可能导致音乐加载失败`

加载问题：一次性加载过多播放器，使网页卡顿。

> 解决办法:计划使用JavaScript实现播放器动态加载，但由于时间和技术储备有限，这部分功能仍在学习中...



> 这是一个学习过程中的项目，还有很多需要改进和完善的地方 💪
